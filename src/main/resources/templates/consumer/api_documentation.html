Authentication

API Key Authentication

Header

All API requests require authentication using your API key. Include it in the Authorization header:

Authorization: Bearer YOUR_API_KEY
Headers
Parameter Type Required Description
Authorization string Yes Bearer token with your API key
Content-Type string Yes application/json

Core Endpoints

Get Battery Health Data

GET
GET /api/v1/battery/health

Retrieve battery state of health (SoH) data with various filtering options.

Query Parameters
Parameter Type Required Description
vehicle_id string No Filter by specific vehicle ID
start_date string No Start date (YYYY-MM-DD)
end_date string No End date (YYYY-MM-DD)
region string No Filter by region
Example Request
curl -X GET "https://api.evdatahub.com/v1/battery/health?start_date=2024-01-01®ion=europe" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
Example Response
{
  "data": [
    {
      "vehicle_id": "EV_7842",
      "timestamp": "2024-03-15T10:30:00Z",
      "state_of_health": 92.7,
      "cycle_count": 124,
      "temperature_avg": 25.4,
      "region": "europe"
    }
  ],
  "pagination": {
    "total": 1500,
    "page": 1,
    "per_page": 100
  }
}

Get Charging Sessions

GET
GET /api/v1/charging/sessions

Retrieve detailed charging session data including duration, energy consumed, and charger types.

Query Parameters
Parameter Type Required Description
charger_type string No AC, DC, or Fast
min_duration integer No Minimum session duration in minutes
max_energy number No Maximum energy consumed in kWh

Get Driving Patterns

GET
GET /api/v1/driving/patterns

Access driving pattern data including mileage, speed profiles, and route efficiency.
